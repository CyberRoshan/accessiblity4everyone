Our Approach

We use a hybrid approach. What does that mean?

A mix of automated and manual testing. We use existing software that reports errors like broken links, missing alt text, missing headings, redundant links, and other structural issues.

Manual testing reveals errors like irrelevant alt text, ARIA issues, zoom errors, keyboard testing, and any other screen reader reviews.
